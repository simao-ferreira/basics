 ## Swagger / OpenAPI
 
 To use swagger/openapi to spring boot add [springdoc](https://springdoc.org/) package.
 
 Configurations can be done through:
 ```
springdoc.swagger-ui.path=swagger-ui
springdoc.swagger-ui.url=/api-docs/swagger.yaml
```
